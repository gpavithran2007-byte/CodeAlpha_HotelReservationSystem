# Hotel Reservation System

A Java console-based Hotel Reservation System developed as part of the CodeAlpha Internship.

## Features
- View available rooms
- Book a room
- Cancel booking
- Simple menu-driven interface

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner

## How to Run

Compile the program:
javac HotelReservationSystem.java

Run the program:
java HotelReservationSystem
